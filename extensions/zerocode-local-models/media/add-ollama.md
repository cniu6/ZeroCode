## Ollama quick setup

1. Install and start Ollama locally.
2. Pull a coding model such as `qwen2.5-coder:7b` or `deepseek-coder-v2`.
3. Or run **ZeroCode: Install Recommended Ollama Model...** to start a pull in the integrated terminal.
4. Run **ZeroCode: Add Ollama Provider**.

ZeroCode will query `http://127.0.0.1:11434/v1` for available models and surface them in chat.
